# Do Better

This is a self-improvement application.

## Targeted Platforms

-ios
-android

## About this app

Gamified Self-improvement (Habit tracker) application.

The app is aimed to help you in one or all of the four categories, exercise, diet, memorization, and self-improvement. The app uses habits as a way to track progress and keep the user on task. Users can create a habit and assign it to a category that matches.

Currently the application is in it's early stages and not all features are implemented. Currently the user sign-up/sign-in, creating habits and habit tracking are currently working, as well as the xp tracking from completing a task.

Daily streaks have been added and soon notifications will be added to remind the user to check-in and when they are close to losing their streak.
